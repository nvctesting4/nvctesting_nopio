Test Page: Intentionally Bad Core Web Vitals

This page is designed to perform poorly so that Core Web Vitals tools (like Lighthouse or Search Console) can show warnings and “failed” reports.
Do not copy this pattern to real pages.

Huge Image Loaded Late

The image below is large and we do not reserve height with CSS. When it loads, the layout will jump (CLS).

Slow Interaction Button (Bad INP)

Click the button below. The JavaScript handler intentionally blocks the main thread for a while to simulate a very slow response to user input.

Why this page exists

  • To trigger LCP issues via a big, late-loading image.
  • To trigger CLS with an injected banner and images without reserved space.
  • To trigger INP problems with a slow click handler.

This is a diagnostic page for nopio.online. It’s safe to crawl, but not meant for normal visitors.
