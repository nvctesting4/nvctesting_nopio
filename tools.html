Redirect Test Page

This page tests how crawlers detect redirected URLs on external websites.

✔ Good: Direct Link (No Redirect)

This link goes straight to the correct page without any redirection.

  • MDN Web Docs (direct, no redirect)

⚠ Bad: External Link That Redirects

This link first redirects to another URL. Crawlers flag this because you should link to the final destination instead.

  • Wikipedia (redirects from HTTP → HTTPS)
  • Bit.ly shortened link (redirects to another site)
